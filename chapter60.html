メソッド定義

Scalaのメソッド定義は次のように書きます。

def methodOne(): String = {
  "hoge" + "foo"
}

メソッド定義の波括弧は必ずしも必要ではありません。

def methodOne(): String = "hoge" + "foo"

戻り値の型も省略できます。 戻り値の型はコンパイラにより自動推論されます。

def methodOne() = "hoge" + "foo"
